PLAN: Build an automated “fresh content engine” for The NFL Edge using GitHub Pages + GitHub Actions.

GOAL:
- Automatically update the homepage iframe with the newest weekly content.
- Avoid editing HTML manually.
- Keep the site static, fast, and fully GitHub‑hosted.

ARCHITECTURE:
1. Store weekly content in a structured folder:
   /content/YYYY/weekX.htm

2. Maintain a JSON file at the repo root:
   latest.json
   {
     "latest": "content/YYYY/weekX.htm"
   }

3. Homepage loads content dynamically:
   <iframe id="contentFrame"></iframe>
   <script>
     fetch("latest.json")
       .then(r => r.json())
       .then(data => {
         document.getElementById("contentFrame").src = data.latest;
       });
   </script>

4. GitHub Action workflow:
   - Trigger on push to main.
   - Detect newest .htm file in /content/.
   - Update latest.json automatically.
   - Commit the update.
   - Trigger GitHub Pages rebuild.

5. Optional enhancements:
   - Generate an archive index JSON.
   - Auto‑rotate older content.
   - Add a “Latest Analysis” badge.
   - Add a dropdown of all weeks.
   - Add season‑based logic (auto‑switch each NFL week).
   - Add sitemap.xml for SEO.
