# PanaGeo(Tech) Score Projection Algorithm 🏈
### Proprietary UFL Game Analytics & Margin Probability Engine

The **PanaGeo(Tech)** algorithm is a high-fidelity scoring model designed to predict offensive efficiency and game volatility in the United Football League (UFL). 

Unlike standard "Power Ratings," PanaGeo utilizes a specific mathematical constant—the **PanaGeo Pi ($Pg\pi$)**—to weight performance against historical win-ratios, identifying the "Surge Factor" in professional spring football.

---

## 1. The Core Variables
The model operates on a per-quarter efficiency basis:
*   **PPQF (Points Per Quarter For):** `(Total Season Points For / Games Played) / 4`
*   **PPQA (Points Per Quarter Allowed):** `(Total Season Points Allowed / Games Played) / 4`

## 2. The PanaGeo Pi ($Pg\pi$) Constants
The algorithm selects a multiplier based on the team's Win Ratio (WR) over the trailing 4-game window (or season-to-date):


| Win Ratio (WR) | Performance Tier | $Pg\pi$ Constant |
| :--- | :--- | :--- |
| **0%** | Sub-Baseline | **1.97531415** |
| **25%** | Defensive/Struggling | **2.031415** |
| **50%** | Pivot Point | **2.131415** |
| **75% - 100%** | Elite/Dominant | **2.31415** |

---

## 3. The Two-Tier Output System

### Tier I: PCPS (PanaGeo Computer Projected Score)
The raw, purely mathematical projection of a team's efficiency in a specific matchup.
*   **Formula:** `([Team_PPQF] + [Opponent_PPQA]) * PgPi`

### Tier II: PAGS (PanaGeo Adjusted Game Score)
The "Real World" score. This applies the **Weighted Variance Adjustment (WVA)** of **0.7314** to account for momentum and team morale.

*   **WR < 50%:** `PCPS - 0.7314` → **Floor** (Round down to nearest whole number).
*   **WR = 50%:** `PCPS` → **Standard Rounding** to nearest whole number.
*   **WR >= 75%:** `PCPS + 0.7314` → **Ceiling** (Round up to nearest whole number).

---

## 4. PPR (PanaGeo Projection Ratio)
The **PPR** is the ultimate indicator of game flow and margin probability. It is calculated by dividing the higher PAGS by the lower PAGS.


| PPR Value | Probability Narrative |
| :--- | :--- |
| **1.00 – 1.20** | **Toss-Up:** High probability of a one-score game. |
| **1.21 – 1.60** | **Comfortable:** High probability of a 8-14 point margin. |
| **1.61+** | **Blowout:** Statistical safety net suggests a multi-score victory. |
